Detailed Description

printa na tela a história final do jogo, o destino final do jogador

essa classe possui uma string para armazenar o nome do jogador e um inteiro para armazenar a escolha que o jogador desejará fazer para salvar uma das pessoas que ele batalhou, além do construtor e 4 funções.
